# 🥚 网站彩蛋指南

这个网站藏了一些有趣的彩蛋，等你来发现！

## 已实现的彩蛋

### 1. 🎮 Konami Code

输入经典的 Konami Code：`↑ ↑ ↓ ↓ ← → ← → B A`
解锁后会触发矩阵雨效果和特殊消息！

### 2. ✨ 鼠标轨迹

移动鼠标时会随机出现渐变色的小光点，增添科技感。

### 3. 🎨 Logo 交互

- 悬停在导航栏的 logo 上会有旋转效果
- 点击头像会有缩放动画

### 4. 🌈 主题切换动画

点击主题切换按钮时，图标会旋转，带来流畅的视觉体验。

## 设计特点

- **深色主题优先**：默认使用深色主题，符合开发者审美
- **等宽字体**：使用 IBM Plex Mono，营造代码感
- **绿紫渐变**：Logo 和强调色使用绿色到紫色的渐变
- **极简留白**：大量负空间，让内容呼吸
- **微交互**：悬停、点击都有细腻的反馈

## 技术实现

- Next.js 14 + TypeScript
- Framer Motion 动画
- CSS Modules
- 自定义 Canvas 动画（矩阵雨）
- 键盘事件监听（Konami Code）
- 鼠标事件监听（轨迹效果）
